# GitHub Integration

You can automatically deploy your Reflect app using GitHub Actions.

<Steps>

### Generate an API Key

```bash
npx reflect keys create my-github-actions-api-key
```

For Permissions select `app:publish` and for Authorized Apps select `(created apps)`.

### Create Secret

In your GitHub repo, go to Settings → Secrets and Variables.

Use "New repository secret" to create a new secret named `REFLECT_API_KEY` with the value of the API key you generated above.

Use "New repository variable" to create a new variable named `REFLECT_APP_NAME` with the value of the name of your Reflect app.

### Create Github Action for Deploying to Production on push to main

In your GitHub repo, go to Actions → New Workflow → set up a workflow yourself.

Paste the following into the editor:

```yaml
name: Deploy Reflect to Production

on:
  push:
    branches:
      - main

env:
  NODE_VERSION: "18.x"

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Set up Node.js
        uses: actions/setup-node@v3
        with:
          node-version: ${{ env.NODE_VERSION }}
          cache: "npm"

      - name: npm install
        run: |
          npm install

      - name: publish
        env:
          REFLECT_AUTH_KEY: ${{ secrets.REFLECT_API_KEY }}
          REFLECT_APP_NAME: ${{ vars.REFLECT_APP_NAME }}
        run: |
          npx reflect publish --app=$REFLECT_APP_NAME --auth-key-from-env=REFLECT_API_KEY
```

### Create Github Action for Deploying Previews for Pull Requests

The following will publish a preview of each pull request to:
`https://${reflect-app-name}-${branch-name}-${reflect-team-name}.reflect-server.net`. Where
`reflect-app-name` is the value you set for you github REFLECT_APP_NAME variable,
`branch-name` is the name of the pull request's branch, and `reflect-team-name` is
the github username you used to log in to Reflect's CLI.

In your GitHub repo, go to Actions → New Workflow → set up a workflow yourself.

Paste the following into the editor:

```yaml
name: Deploy Reflect to Preview

on:
  pull_request:
    branches: [main]

env:
  NODE_VERSION: "18.x"

jobs:
  preview:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Set up Node.js
        uses: actions/setup-node@v3
        with:
          node-version: ${{ env.NODE_VERSION }}
          cache: "npm"

      - name: npm install
        run: |
          npm install

      - name: publish
        env:
          REFLECT_AUTH_KEY: ${{ secrets.REFLECT_API_KEY }}
          REFLECT_APP_NAME: ${{ vars.REFLECT_APP_NAME }}
        run: |
          npx reflect publish --app=${REFLECT_APP_NAME}-${GITHUB_HEAD_REF/\//-} --auth-key-from-env=REFLECT_API_KEY
```

</Steps>
